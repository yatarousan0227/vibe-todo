LLM Provider Configuration

design_id: 001-vibetodo-llm-provider-adapter — REQ-005, REQ-006
このパネルは UI 画面ではなく、LLMProvider adapter の環境変数スキーマを開発者向けに可視化するものです。

プロバイダー選択 (CFG-001)

変数名 必須 許容値 説明
LLM_PROVIDER 必須 openai | anthropic | azure_openai 使用する LLM プロバイダーを選択する。未設定または無効値の場合は起動を中断する。

OpenAI 設定 (CFG-002) — LLM_PROVIDER=openai のとき有効

変数名 必須 デフォルト 例 / 説明
OPENAI_API_KEY 必須 — sk-... 形式の API キー
OPENAI_MODEL 必須 — gpt-4o
OPENAI_API_BASE_URL 省略可 https://api.openai.com/v1 proxy や互換 API を使う場合に設定する

Anthropic 設定 (CFG-003) — LLM_PROVIDER=anthropic のとき有効

変数名 必須 デフォルト 例 / 説明
ANTHROPIC_API_KEY 必須 — sk-ant-... 形式の API キー
ANTHROPIC_MODEL 必須 — claude-sonnet-4-6
ANTHROPIC_API_BASE_URL 省略可 https://api.anthropic.com proxy を使う場合に設定する

Azure OpenAI 設定 (CFG-004) — LLM_PROVIDER=azure_openai のとき有効

変数名 必須 例 / 説明
AZURE_OPENAI_API_KEY 必須 Azure OpenAI リソースの API キー
AZURE_OPENAI_ENDPOINT 必須 https://my-resource.openai.azure.com
AZURE_OPENAI_DEPLOYMENT_NAME 必須 gpt-4o-deployment
AZURE_OPENAI_API_VERSION 必須 2024-02-01

LLMProvider Port Contract (REQ-001)

interface LLMProvider {
  generateText(prompt: string, options?: LLMCallOptions): Promise<LLMResult>;
}

type LLMCallOptions = {
  maxTokens?: number;
  temperature?: number;
};

type LLMResult = {
  text: string;
  usage?: { promptTokens: number; completionTokens: number };
};

type LLMError = {
  code: "AUTH_ERROR" | "RATE_LIMITED" | "NETWORK_ERROR" | "PROVIDER_ERROR";
  message: string;
  providerName: string;
};
